# Getting Started

- Fork this repo
- Clone on your local machine

```
git clone https://github.com/asiya00/Machine-Learning.git
```
- Navigate to project directory.
```
cd Machine-Learning
```
- Add your contribution
```
git add .
```
- Commit your changes.

```markdown
git commit -m "Relevant message"
```
- Then push 
```
git push
```
